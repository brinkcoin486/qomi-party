# qomi-party
RIG WALLET BRIDGE – README
But du programme
RIG WALLET BRIDGE est un lanceur simple qui ouvre une interface HTML unique pour gérer les actions crypto depuis un seul point d'accès.
Le but n'est pas de gérer une seule monnaie, mais de centraliser plusieurs directions de travail dans une seule interface.
Pourquoi il est multidirectionnel
L'interface est pensée pour fonctionner comme un point de passage entre plusieurs éléments :
différentes crypto-monnaies,
différents wallets,
différents réseaux,
différents pools,
différents liens et outils externes.
Chaque zone peut pointer vers une destination différente sans casser l'ensemble.
Le lanceur C++ ouvre simplement la page source. Toute la logique visible, toute l'organisation et toute la souplesse de navigation sont contenues dans la page HTML.
Pourquoi il peut gérer toutes les crypto-monnaies
Le système n'est pas limité à une seule liste figée.
Il peut accueillir toute crypto-monnaie tant qu'on peut définir au minimum :
un nom,
une adresse,
un réseau,
une destination ou un outil associé.
Cela permet de suivre aussi bien des monnaies connues que des réseaux plus rares, sans devoir réécrire tout le projet.
Réduction des fautes de frappe
Le projet sert aussi à éviter les erreurs manuelles.
Au lieu de retaper les informations à chaque fois, l'utilisateur travaille depuis une base déjà écrite, déjà relue et déjà organisée.
Cela réduit :
les fautes de frappe,
les erreurs de copier-coller,
les oublis d'adresse,
les confusions entre wallets, coins et réseaux.
